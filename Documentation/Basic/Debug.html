Debug Macro

Definition

Debug(message)

Input parameters

message
This text will sent to the debug output.

Description

The propose of this macro is help debugging, sending messages to the debug output, on POSIX systems the output will be stderr. On Windows systems the debug output can be seen using the program DebugView which is part of SysInternals or if the program is being debugged within Visual Studio then the debug output is seen in the Output panel.

This is a macro and does not belong to the Grok namespace.

This macro can be disabled if the NDEBUG macro is defined in the compiler command line or in the project properties under preprocessor definitions.

Example

The following code is an example.

Debug("This code works!");

If the NDEBUG macro is not defined then the debug output will show.

This code works!

Requirements

#include <Basic/Debug.h>

Grok::DebugMessage Function

Definition

namespace Grok
{
   void DebugMessage(const char* message) throw();
}

Input parameters

message
This text will sent to the debug output.

Description

The propose of this function is help debugging, sending messages to the debug output, on POSIX systems the output will be stderr. On Windows systems the debug output can be seen using the program DebugView which is part of SysInternals or if the program is being debugged within Visual Studio then the debug output is seen in the Output panel.

The difference between this function and the Debug macro is that this function can not be disabled by the NDEBUG macro.

Example

The following code is an example.

DebugMessage("This code works!");

The debug output will show.

This code works!

Requirements

#include <Basic/Debug.h>

Grok::DebugStop Function

Definition

namespace Grok
{
   void DebugStop() throw();
}

Description

This function stops the execution of the program. If the program is being run inside a debugger then the execution stops and the control is returned to the debugger. If the program runs outside of a debugger then the program is ended.

This function can not be disabled by the NDEBUG macro.

Requirements

#include <Basic/Debug.h>
